Plot a set of markers on the display.

Parameters
xData source for the x axis
yData source for the y axis
cData source for the color of the markers. Note that the final color is a function of the color map set via set_color_map and the value from c
sData source for the size of the markers. The size is normalized. Hence a size of 1.0 will draw a marker as big as the display.
windowOptional drawing region used to draw the line. Default is fullscreen
optsOptional scatter option. Default is the string "o", which will draw the marker as a circle. Multiple character can be used to create composite markers, e.g. "xo". All the possible marker shapes are "eps>^*x+o."
